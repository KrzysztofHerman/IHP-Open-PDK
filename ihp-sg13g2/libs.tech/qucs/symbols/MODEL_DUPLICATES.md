# Qucs symbol model value duplications

The entries below list `<DefaultModel>` and `<SpiceModel>` values that appear in more than one symbol XML file under `ihp-sg13g2/libs.tech/qucs/symbols`. File names are sorted alphabetically within each group.

- **D_IHP**: `dantenna.xml`, `diodevdd_2kv.xml`, `diodevdd_4kv.xml`, `diodevss_2kv.xml`, `diodevss_4kv.xml`, `dpantenna.xml`
- **MOS_IHP**: `nmoscl_2.xml`, `nmoscl_4.xml`, `sg13_hv_nmos.xml`, `sg13_hv_pmos.xml`, `sg13_lv_nmos.xml`, `sg13_lv_pmos.xml`
